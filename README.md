# Telegram Bot


### Installing package

```bash
npm install
```

### Setup env in env.example

```bash
BOT_TOKEN=
JWT_PINATA_CLOUD=
NEXT_PUBLIC_RELAYER_ACCOUNT_ID_NEAR_MAINNET=
RELAYER_PRIVATE_KEY_NEAR_MAINNET=
```

### Setup API telegram bot

* [API](./api/README.md)

### Run project 

```bash
npm start
```
* Open telegram bot and enter start to begin
